*   **AWS SCT**: Schema Conversion Tool
*   typically used for heterogeneous migrations where you’re moving from one type of database to another
*   **CDC:** Change Data Capture
*   **AWS DMS:** Database Migration Service
*   **Amazon QLDB**: Quantum Ledger Database
*   **EBS:** Elastic Block Storage
*   **RPO (Recovery Point Objective):** How much data can you afford to lose? (time between backups)
*   **RTO (Recovery Time Objective):** How quickly must you recover? (downtime tolerance)
*   **KCL:** Amazon Kinesis Client Library
*   **DAX:** DynamoDB Accelerator
*   **RBAC:** role-based access control
*   **NACL**: Network Access Control List
*   PITR: Point-in-Time Recovery
*   **Systems Manager Automation service** is primarily used to simplify common maintenance and deployment tasks of Amazon EC2 instances and other AWS resources
*   **AWS Compute Optimizer** simply helps you to identify the optimal AWS resource configurations, such as Amazon Elastic Compute Cloud (EC2) instance types, Amazon Elastic Block Store (EBS) volume configurations, and AWS Lambda function memory sizes. It is not capable of providing recommendations to improve your workloads based on AWS best practices.
*   **AWS Well-Architected Tool** provides a consistent approach for evaluating your cloud architecture and implementing designs that will scale with your application needs over time  
    
*   RMAN: Recovery Manager  
    
*   **TDE:** transparent data encryption: is primarily used to encrypt stored data on your DB instances running Microsoft SQL Server and not the data that is in transit
*   **IAM**: IAM database authentication is only supported in MySQL and PostgreSQL database engines
*   **Amazon RDS Standby Replicas**: is used in Multi-AZ deployments, 
*   **AWS Read Replica** primarily provides an asynchronous replication
*   **Amazon DynamoDB Read Replica**: uses global tables to replicate data across multiple AWS Regions
*   **Multi-AZ deployments** feature is mainly used to achieve high availability and failover support for your database
*   **STS** is used to send temporary tokens for authentication
